##  开发步骤

开发一个Java程序要经历以下几个步骤

1. **编写**
	1. 编写代码即可
2. **编译**
	1. 使用 `javac 文件名`
3. **运行**
	1. 使用`java 编译过的文件名`
	2. 不要带后缀名否则运行失败

## 自动类型转换

**类型范围小的变量，可以直接赋值给类型大的变量**

比如： byte类型可以直接赋值给int类型

**自动类型转换在计算机中的执行原理**
- a是byte 占用1个字节 也就是8位
- b是int类型 占用4个字节 也就是32位
- 所以a的范围比b的范围小可以赋值
![[Pasted image 20260428143143.png]]

**自动类型转换的其他形式**
- 箭头左侧的都可以赋值给右侧的
- long(8字节)可以赋值给float(4字节)是因为float的数值范围比long大
- char可以赋值给int
	- 从 范围看 char是2字节 int是4字节 chat比int小
	- 从类型上来看 char它存储的是 **Unicode** 字符编号 编号也是整型

```
byte -> short -> int -> long -> float -> double

char -> int
```

**代码**
```java
public class TypeConversionDemo1 {
    public static void main(String[] args) {
        byte a = 12;
        int b = a; // 这里发生了自动类型转换
        System.out.println(a); // 12
        System.out.println(b); // 12

        int i =999;
        double j = i;
        System.out.println(j); // 999
        System.out.println(i); // 999.0


        char c = 'b'; // 编码是98
        int it = c;
        System.out.println(c); // b
        System.out.println(it); // 98 这里输出因为类型是int 所以输出的是98
    }
}
```

## 表达式自动类型转换

**在表达式中，小范围类型的变量，会自动转换成表达式中较大范围的类型，再参与运行**

```
byte、short、char -> int -> long -> float -> double
```

**注意**
 - 表达式的最终结果类型由表达式中的**最高类型决定**
 - 在表达式中，byte、short、char直接转换成`int`进行运算的
**代码**
```java
public class TypeConversionDemo2 {
    public static void main(String[] args) {
        byte a = 10;
        short b = 20;
        int c = 30;
        long d = 40;
        // int result = a + b + c + d; 报错 最大类型是long
        long result = a + b + c + d;


        // long result2 = a + d + 3.14; // 报错最大类型是double 3.14的类型是double
        double result2 = a + d + 3.14;


        byte a1 = 10;
        byte a2 = 20;
        // byte a3 = a1 + a2; // 报错 byte、short、char直接转换成int进行运算的
        int a3 = a1 + a2;
    }
}

```

## 强制类型转换

- 强行将类型范围大的变量、数据赋值给类型范围小的变量
```
数据类型 变量2 = (数据类型) 变量1、数据
```

**转换原理**
- 直接从范围大的类型中获取符合小类型位数的值放进去
- 对于强制转换的位数超过的会进行截取
![[Pasted image 20260428160218.png]]

**注意**
- 强制类型转换可能会造成**数据溢出**
- 浮点型强制转换成整型，直接**丢掉小数部分**，保留整数返回


**代码**
```java
package com.puta99.type;

public class TypeConversionDemo3 {
    public static void main(String[] args) {
        int a = 20;
        // byte b = a; // 报错 大范围类型的伴侣不能直接赋值给小范围类型的变量
        byte b = (byte) a; // 强制转换

        int i = 1500;
        byte j = (byte) i; // 强制转换类型可能出现数据溢出(失真)
        System.out.println(i); // 1500
        System.out.println(j); // -36


        double score = 99.5;
        int number = (int) score; // 浮点类型小数强制转换成整数直接丢失小数部分，返回整数部分
        System.out.println(number); // 99
    }
}

```
# 运算符

## 基本运算符

```
+ - * / %
```

**注意**
- 两个整型如果相除是小数，那么会被强制转成整数
- `+`还能作为连接符，口诀：能算就算，不能算就在一起

**代码**
```java
public class OperatorDemo1 {
    public static void main(String[] args) {
        int a = 5;
        int b = 2;
        System.out.println(a + b); // 7
        System.out.println(a - b); // 3
        System.out.println(a * b); // 10
        // 因为a和b两个类型中最大的范围是int 虽然正确结果是2.5 但是会转成int 那么最终就是2
        System.out.println(a / b); // 2
        System.out.println(a % b); // 1 取余

        // a/b 如果想要得到2.5
        // System.out.println(a / b * 1.0); // 2.0 不能这样 因为a/b先执行变成2.0了
        System.out.println(1.0 * a / b); // 2.5
        
         // + 运算符
        int c = 5;
        System.out.println(c + "a"); // 5a
        System.out.println(c + 5); // 10
        System.out.println(c + 'a'+ "哈哈"); // 102哈哈  'a'是char存储的是编码97 所以可以计算结果是102
    }
}

```

## 自增、自减
- 只能操作变量 ，不能操作字面量

**单独使用**
- 单独使用++、--在变量前后效果是一样的
```java
public class OperatorDemo3 {
    public static void main(String[] args) {
        int a = 10;
        a++;
        ++a;
        System.out.println(a);

        int b = 10;
        b--;
        --b;
        System.out.println(b); // 8
    }
}
```

**非单独使用**
- ++、--如果不是单独使用(如在表达式中、或者同时有其它操作)，放在变量前、后有明显的区别

**变量前（先加后用）**
- 放在变量前面，先对变量进行+1、-1操作，在拿变量的值进行运算
```java
int a = 10;
int rs = ++a
System.out.println(rs); // 11
```
**变量后（先用后加）**
- 放在变量后面，先拿变量的值进行运算，在对变量的值进行+1、-1
```java
int a = 10;
int rs = a++
System.out.println(rs); // 10
```

## 拓展赋值运算符

- 扩展的赋值运算符隐含了强制类型转换

| 符号  | 用法   | 作用    | 底层代码形式             |
| --- | ---- | ----- | ------------------ |
| +=  | a+=b | 加后赋值  | a = (a的类型)(a + b); |
| -=  | a-=b | 减后赋值  | a = (a的类型)(a - b); |
| *=  | a*=b | 乘后赋值  | a = (a的类型)(a * b); |
| /=  | a/=b | 除后赋值  | a = (a的类型)(a / b); |
| %=  | a%=b | 取余后赋值 | a = (a的类型)(a % b); |

**举例**
```java
int a = 10;
byte b = 1;
a += b;

// 等价于下面的写法
// 首先a+b的值
// 然后把它们的值强制转换成a(int)的类型
// 最后把转换后的值赋值给a
a = (int) (a + b);


// 可以避免自带强制类型转换  
byte t1 = 100;  
byte t2 = 10;  
// byte t3 = t1 + t2; 报错 因为在表达式中，byte、short、char直接转换成`int`进行运算的  
t1 += t2; // 这样就能避免转换成int了 因为等价于 t1 = (byte) (t1 + t2);
```

## 逻辑运算符

- 在java中&、|，无论左边是fasle或true右边都要执行
- 由于&&、||运算效率更高、在开发中用的更多

| **符号** | **叫法** | **例子**           | **运算逻辑**                                          |
| ------ | ------ | ---------------- | ------------------------------------------------- |
| **&**  | 逻辑与    | `2 > 1 & 3 > 2`  | 多个条件必须都是 **true**，结果才是 **true**；否则是 false。        |
| **\|** | 逻辑或    | `2 > 1 \| 3 < 5` | 多个条件中只要有一个是 **true**，结果就是 **true**。               |
| **!**  | 逻辑非    | `!(2 > 1)`       | 取反                                                |
| **^**  | 逻辑异或   | `2 > 1 ^ 3 > 1`  | 前后条件的结果相同，就直接返回 **false**；前后条件的结果不同，才返回 **true**。 |
| &&     | 短路与    | 2>10&&3>2        | 判断结果同"&"，左边位fasle，右边不执行                           |
| \|\|   | 短路或    | 2>1\|\|3<5       | 判断结果同"\|"，左边位true，右边不执行                           |

## Random

nextInt(start,end)带这种区间范围的是从jdk17才开始支持的

通过**减加法**来实现这种范围
```java
public class RandomDemo2 {
    public static void main(String[] args) {
        Random r = new Random();
        // 区间是 从jdk17开始支持的
        // 生成2-10的随机数
        // int number = r.nextInt(2,10);
        // System.out.println("随机生成了：" + number);

        // 自己实现区间
        // 基于减加法
        // r.nextInt 是从0开始的
        // 假设我们需要的是2-10的范围
        // 先往0靠拢 都减去2 变成 0-8
        // 这样r.nextInt(9) 这样就能得到了 因为不包含9
        // 因为之前减去了2 所有再获得值后加回来
        // 最终就变成 r.nextInt(9)+2
        int number1 = r.nextInt(9)+2;
        System.out.println("随机生成了：" + number1);

    }
}
```

# 方法
- java中没有函数的概念，全都是方法，其实就是函数，只不过java都函数都在类中，所以叫方法

## 基本使用

```
修饰符 返回值类型 方法名(行参列表) {
	方法体(实际内容)
	return xxx;
}

# 调用
方法名(参数列表)
```

**注意**
- 如果方法不需要返回数据，返回值类型必须声明成**void**，此时方法内部不能使用**return**返回数据

**代码**
```java
public class MethoDemo1 {
    public static void main(String[] args) {
        System.out.println(getMax(10, 30));
    }


    // 比较大小的方法
    public static int getMax(int a, int b) {
        return a > b ? a : b;
    }
    
    // 没有返回值和参数的方法
    public static void noReturn() {
        System.out.println("123");
    }
}

```

## 注意事项
- 方法在类中的位置不重要，但一个方法不能定义在另一个方法里面
- 如果方法不需要返回数据，返回值类型必须声明成**void**，此时方法内部不能使用**return**返回数据
- 方法有返回类型，必须使用return语句返回对应的类型
- 调用方法时，入参的数据必须严格匹配方法的行参

	- 栈是先进后出的特性
	- 可以保证一个方法调用完另一个方法后，可以回来(拿到返回值)
## 参数传递

- 传参都是**值传递**
- 基本类型传入的是**副本值**，引用类型传入的是**地址**

**基本类型**
- 传递参数时，传输的是实参的副本
- 这一点同js一样

```java
public class MethoDemo1 {
    public static void main(String[] args) {
       int a = 10;
       // 这里传入的a 实际上是对a复制了一份
       test(a)
       System.out.println(a) // 10
    }


    public static int test(int a) {
	    // 传入的是a的副本 这里的更改不会影响到传入的a
	    a = 20;
        return a;
    }

}

```

**引用类型**
- 传递参数时，传输的是实参的地址，所以改动会有副作用
- 这一点同js一样
```java
public class MethodParamDemo1 {
    public static void main(String[] args) {
        int[] arr = {1, 2, 3, 4};
        // 传入arr的地址
        change(arr);
        System.out.println(arr[0]); // 10
    }

    public static void change(int[] arr) {
```

## 实体类

 **就是一种特殊形式的类，需要符合下面的条件**
- 1.这个类中的**成员变量**都要**私有**，且要对外提供相应的getxxx和setxxx方法
- 2.类中必须要提供一个公共的**无参数构造器**


**使用场景**
- 实体类的对象只负责数据的存取
- 而对数据的处理要交给其他对象来完成，以实现数据和数据业务处理相分离


**作用**
- 实体类的对象本身只负责存取对象的数据
- 对于数据的业务处理应该交由另一个类的对象专门处理
- 这是软件设计中很流行的一种设计思想**分层思想**

```java
public class StringDemo1 {
    public static void main(String[] args) {
        // 方式一： 直接字面量的形式创建(推荐使用)
        String name = "张三";
```

```java
public class StringMethodsDemo2 {
    public static void main(String[] args) {
        String s1 = "哈我哈哈你好";

        // length
        System.out.println(s1.length()); // 5

        // chartAt
        System.out.println(s1.charAt(1)); // 我

        System.out.println((s1.toCharArray())); // 哈我哈哈你好

        // equals 比较的是内容
        // == 比较的是地址而不是内容
        // Java 的字符串常量池机制：相同的字符串字面量会指向同一个对象。
        // 但是如果使用构造器创建的字符串 那么不会指向同一个对象
        String s2 = new String("哈我哈哈你好");
        String s3 = "哈我哈哈你好";
        System.out.println(s1 == s2); // false
        System.out.println(s1 == s3); // true
        System.out.println(s1.equals(s2)); // true

        // equalsIgnoreCase
        String s4 = "abCd";
        String s5 = "abcd";
        System.out.println(s4.equalsIgnoreCase(s5)); // true

        // substring
        System.out.println(s1.substring(0, 2)); // 哈我

        // replace 因为返回的是字符串所以可以使用链式调用
        String res = s1.replace("哈","嘻").replace("你","我");
        System.out.println(res); // 嘻我嘻嘻我好

        // contains
        System.out.println(s1.contains("哈")); // true

        // startWith
        System.out.println(s1.startsWith("嘻")); // false

	    // split
        String s6 = "张三，李四，王五";
        // 返回的是一个String[] 只不过是一个地址
        // 类似： String[] s = {"张三", "李四", "王五"};
        System.out.println(s6.split("，"));
```

### 案例

#### 案例一

**有问题**
```java
public class ArrayListDemo2  {
    public static void main(String[] args) {
        ArrayList<String> list = new ArrayList();

        list.add("香蕉");
        list.add("枸杞1");
        list.add("枸杞2");
        list.add("草莓");

        System.out.println(list); // [香蕉, 枸杞1, 枸杞2, 草莓]

        for (int i = 0; i < list.size(); i++) {
            String name = list.get(i);
            if (name.contains("枸杞")) {
                list.remove(i);
            }
        }
        // 没有删除干净
        // 原因是
        // 当i==0 香蕉不符合 所以不会删除
        // 当i==1 枸杞1符合 删除枸杞1 后面的元素往前补位 此时list是 [香蕉, 枸杞2, 草莓]
        // 当i==2 草莓不符合 所以不会删除
        System.out.println(list); // [香蕉, 枸杞2, 草莓]


    }
}
```

**解决**
```java
for (int i = 0; i < list.size(); i++) {  
    String name = list.get(i);  
    if (name.contains("枸杞")) {  
        list.remove(i);  
        // 既然后续元素会往前补位 那么让i-- 这样就可以对上了  
        i--;  
    }  
}

// 输出正常
System.out.println(list); // [香蕉, 草莓]


// 或者采用倒序的方式遍历
// [香蕉, 枸杞1, 枸杞2, 草莓]
// i == 3 草莓不符合 不删除
// i == 2  枸杞2符合 删除。此时list是[香蕉, 枸杞1, 草莓] 草莓补位 但是不影响因为是倒序
// i == 1 枸杞1符合删除 此时list是[香蕉, 草莓] 
for (int i = list.size() - 1; i > 0; i--) {  
    String name = list.get(i);  
    if (name.contains("枸杞")) {  
        list.remove(i);  
    }  
}  
System.out.println(list);// [香蕉, 草莓]
```
