# 面向对象

## static
- 叫静态，**可以修饰成员变量、成员方法**


### 成员变量
- 访问自己类中的类变量可以**省略类名**，访问别的类的静态变量不能省略类名


**成员变量按照有无static修饰，分为两种**
- **类变量**(静态变量)：有static修饰，在内存中只会存在一份，会被类和类的实例共享
	- **不推荐使用实例访问以及通过实例修改**
- **实例变量**：无static修饰，只能实例对象访问


**代码**
Test.java
```java
public class Test {
    public static void main(String[] args) {
        // static修饰成员变量的用法

        // 静态变量访问方式
        // 方式1： 类名.静态变量访问
        Student.name = "袁华";
        System.out.println(Student.name); // 袁华

        // 方式2: 对象.静态变量访问(不推荐)
        Student s1 = new Student();
        s1.name = "马冬梅";
        System.out.println(s1.name); // 马冬梅


        // 实例变量访问方式
        // 对象.实例变量
        s1.age = 12;
        System.out.println(s1.age); // 12

    }
}
```
Student.java
```java
public class Student {
    // 静态变量 有static修饰，属于类持有，在内存中只加载一次，类本身和它的实例都能访问
    static String name;

    // 实例变量，无static修饰 只能实例访问
    int age;
}
```

### 成员方法
- 访问自己类中的类方法可以**省略类名**，访问别的类的静态方法不能省略类名

**成员方法按照有无static修饰，分为两种**
- **类方法**(静态方法)：有static修饰
- **实例方法**：无static修饰，只能实例对象访问

**代码**
Test.java
```java
public class Test {

    public static void main(String[] args) {
        // 类方法访问方式
        // 方式1: 类名.方法名
        Student.printHello();

        // 方式2 对象.方法名（不推荐）
        Student s1 = new Student();
        s1.printHello();

        // 实例方法访问
        // 对象.方法名
        Student s2 = new Student();
        s2.printScore();
    }
}
```

Student.java
```java
public class Student {
    double score;

    // 静态方法 通过static修饰
    // 当前类和他的所有实例对象可以访问
    public static void printHello() {
        System.out.println("hello");
    }

    // 实例方法 没有static修饰
    // 只能实例对象访问
    public void printScore() {
        System.out.println("分数：" + score);
    }
}
```

#### 使用场景

- 类方法最常见的应用场景就是做**工具类**，能提高代码复用
- 工具类没有创建对象的需求，**建议将工具类的构造器私有化**

**工具类为什么不用实例方法**
- 实例方法需要创建对象调用，工具类只是为了调用方法，如果创建对象会占用内存浪费资源

**代码**
- 如果多处都需要用到验证码这个功能只需要调用VerifyCodeUtils即可
- 不需要每个用到的地方都重新手写一下实现
```java
public class Test {

    public static void main(String[] args) {
      VerifyCodeUtils.code(4);
    }
}
```

VerifyCodeUtils.java
```java
public class VerifyCodeUtils {

	// 私有化工具类构造器 外界可以直接通过类调用 不需要实例对象
	// 因为类会默认有一个无参构造器，如果要私有那么需要手写出来用private修饰
	private VerifyCodeUtils() {}
	
	// 生成验证码的方法
	public static String code(int len) {
		String code = "xxx"
		// 执行生成验证码的逻辑
		
		return code
	}
}
```

### 注意事项

- 1. 静态方法中只能访问静态变量和方法
- 2. 实例方法中可以访问 实例和静态的 方法和变量
-  this只能在实例方法中使用
```java
public class Test {
    // 静态变量
    public static String name;
    // 实例变量
    public int age;
    // 静态方法
    public static void printName() {
        System.out.println(name);
    }

    public static void main(String[] args) {

    }

    // 1. 静态方法中只能访问静态变量和方法
    public static void testStatic() {
        System.out.println(name); // 可以访问静态变量
        // System.out.println(age); 报错 不能访问实例变量
        // this.age; 报错 静态方法不能有this
    }

    // 2. 实例方法中可以访问 实例和静态的 方法和变量
    // 3. this只能在实例方法中使用
    public void testInstance() {
        System.out.println(name);
        System.out.println(age);
        printName();
        System.out.println(this.age);
    }
}
```
### 代码块
- 代码块是类的5大成分之一(成员变量、构造器、方法、代码块、内部类)


**按照有无static代码块分为两种**

**静态代码块**
- **格式**： static {}
- **特点**：类加载时自动执行，由于类只会加载一次，所以静态代码块也**只会执行一次**
- **作用**：完成类的初始化，例如对类变量的初始化赋值
```java
public class CodeTest1 {
    static ArrayList<String> names = new ArrayList<>();

    public static void main(String[] args) {
        System.out.println("main执行了");
    }

    // 静态代码块
    // 用的比较少 了解即可 方便看别人的源码
    static {
        names.add("hhhh"); // 初始化
        System.out.println("static代码块与类一起加载，比main先执行");
    }
}

```
**实例代码块**
- **格式**： {}
- **特点**：每次创建对象实例时，执行实例代码块，**并在构造器之前执行**
- **作用**：和构造器一样，都是用来完成对象的初始化的，例如对 实例变量进行初始化赋值
```java
public class CodeTest2 {
    public static void main(String[] args) {
        // 每次实例新的对象都会调用一次实例代码块和构造器
        CodeTest2 t1 = new CodeTest2();
        CodeTest2 t2 = new CodeTest2();
    }
    
    public CodeTest2() {
        System.out.println("我在实例代码之后执行");
    }

    // 实例代码块
    // 每次创建对象时都会执行一次，且在构造器之前执行
    // 理解即可
    {
        System.out.println("实例代码块：我在实例代码之前执行");
    }
}
```

### 单例模式

- 确保一个类只有一个对象
- 单例模式有很多种


**饿汉单例**
- 拿到对象的时候，对象早已创建好了
```java
public class Single {
    // 2. 创建一个静态变量来存放对象
    // 确保只有一个类的实例
    // 这个静态变量要私有化 防止外部可以更改
    // 外部使用的时候对象已经创建好了(饿汉单例)
    private  static Single single = new Single();

    // 1. 私有化构造器
    private Single () {}

    // 3. 定一个一个静态方法 供外部初始化
    public static Single getInstance() {
        return single;
    }
}
```

**懒汉单例**
- 只有拿对象时，才开始创建对象

```java
public class Single2 {
    // 2. 定义一个静态方法记住类的一个唯一对象
    // 因为是懒汉单例所以不会在类创建的时候就初始化对象
    private static Single2 single2;

    // 1. 私有化构造器
    private Single2() {}

    // 3. 定义一个静态方法 供外部初始化
  public static Single2 getInstance() {
        if (single2 == null) {
            single2 = new Single2();
        }

        return  single2;
  }
}
```

## 继承
- 使用`extends`关键字完成继承
- A类称为父类(基类、超类)
- B类称为子类(派生类)

```java
public class B extends A { }
```


**特点**
- **子类中**能继承父类的**非私有成员**(成员变量、成员方法)
- 一旦继承后，子类的对象是由子类和父类等多张设计图共同完成的，但是子类对象能访问什么，还是要看权限(public/private)
	- 子类对象中其实是有父类或者其他设计图的变量、方法的，只不过访问要看是否有权限

**代码**
Test.java
```java
public class Test {
    public static void main(String[] args) {
        // b对象只能访问A类、B类中有权限的
        B b = new B();
        b.afn2();
        b.bfn2();
        System.out.println(b.b1);
        System.out.println(b.a1);
    }
}
```
A.java
```java
public class A {
    public  int a1;
    private int a2;
    private void afn1() {}
    public void afn2() {}
}
```
B.java
```java
public class B extends A {
    public int b1;
    private void bfn1() {
        // 只能访问父类中非私有的成员变量、成员方法
        afn2();
        System.out.println(a1);
    }
    public void bfn2() {}
}

```

### 权限修饰符
- 就是用来限制类的成员(成员变量、成员方法、构造器、代码块等等)能够被访问的范围

- **private**：只能本类中访问
- **缺省**(不写):同一个包的类、本类中访问
- **protected**:同一个包的类、本类、子孙类中访问
- **public**:任意位置访问
```java
public class Fu  {
    // 1. private修饰的 只能在当前类中访问
    private void printMethod() {
        System.out.println("private");
    }

    // 2. 缺省(没有)修饰符的 ：同一个包的类、本类中访问
    void  method() {
        System.out.println("缺省");
    }

    // 3. protected修饰的：同一个包的类、本类、子孙类中访问
    protected void  protectedMethod() {
        System.out.println("protected");
    }
    
    // 4. public修饰的任意位置都能访问
    public void publicMethod() {
        System.out.println("public");
    }
}
```

### 单继承
- java是**单继承**的，不支持多继承(A extends B, C)
	- 一个类只能继承一个直接父类
- 但是支持多层继承(A extends B, B extends C, C extends E)

### Object类
- object类是java所有类的祖宗类，所有的类都是它的子类或者子孙类

### 方法重写
- 当子类觉得父类中的某个方法不好用，或者无法满足自己的需求时，子类可以重新写一个，**方法名称、参数列表一样**的方法，去覆盖父类的这个方法
- 重写后，方法的访问，java会遵循就近原则

**注意**
- 使用**Override注解**，它可以指定java编译器，检查我们方法重写的格式是否正确，代码的可读性也会更好
- 子类重写父类方法时，访问权限必须大于等于父类该方法的权限(public>protected>缺省)
- 重写的方法返回值类型，必须与被重写的方法的返回值类型一样，或者范围更小
- 私有方法、静态方法不能被重写

**代码**
Cat.java
```java
public class Cat extends Anima {
    public static void main(String[] args) {
        Cat c = new Cat();
        // run方法重写过 java遵循就近原则访问
        // 因为Cat中有所以会访问 Cat的run
        c.run();
    }
    
    @Override
    // 1. 重写的run方法 权限必须大于或等于 父类的run的权限
    // 2. 重写的run的返回值类型， 必须和父类的run的返回值一样，或者比它的返回值范围更小
    void run() {
        System.out.println("父类的run不好 我重新写");
    }
}

```

Anima.java
```java
public class Anima {
    void run() {
        System.out.println("跑起来");
    }
}
```
